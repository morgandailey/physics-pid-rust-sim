# 基於 Rust 之即時物理模擬與 PID 控制系統

### 🛠️ 跨領域實作：機械動力學與資工系統程式的結合

這是一個使用 **Rust 語言** 開發的即時物理模擬引擎。專案核心在於模擬機械工程中經典的「彈簧-質量-阻尼系統」，並透過 **PID 控制演算法** 達成精確的位置控制。

## 🧮 數學模型與物理引擎
本專案的核心是一個 **二階常微分方程 (ODE)** 的數值解算器：

$$m\ddot{x}(t) + c\dot{x}(t) + kx(t) = F_{control}(t)$$

* **數值積分**：使用 **歐拉法 (Euler Method)** 將連續的時間微分離散化，讓系統能在 10ms 的步長下進行高頻率運算。
* **物理參數**：模擬了一個 1kg 的物體在具備阻尼係數與彈簧常數的環境下，受外力驅動的反應。

## ⚙️ PID 控制邏輯
為了讓物體能穩定移動到目標位置（Target Setpoint），我實作了 PID 控制器：
* **P (比例)**：消除當前誤差。
* **I (積分)**：消除系統的靜態誤差（Static Error）。
* **D (微分)**：預測誤差趨勢，減少系統的超越量（Overshoot）並增加穩定性。

## 🦀 為什麼選擇 Rust？
在控制系統與即時模擬領域，效能與穩定性至關重要：
1. **記憶體安全**：Rust 的所有權機制能確保在處理高頻控制迴圈時，不會發生記憶體洩漏或指針錯誤。
2. **零成本抽象**：提供高階的抽象語法，同時保有與 C 語言同等級的執行效能。
